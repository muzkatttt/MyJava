## Домашнее задание к семинару 4 
### Обработка исключений
### 1. Создать программу управления банковским счетом (Account).

Программа должна позволять пользователю вводить начальный баланс 
счета, сумму депозита и сумму снятия средств. 
При этом она должна обрабатывать следующие исключительные ситуации:
1) Попытка создать счет с отрицательным начальным балансом должна вызывать исключение IllegalArgumentException с соответствующим сообщением.
2) Попытка внести депозит с отрицательной суммой должна вызывать исключение IllegalArgumentException с соответствующим сообщением.
3) Попытка снять средства, сумма которых превышает текущий баланс, должна вызывать исключение InsufficientFundsException с сообщением о недостаточных средствах и текущим балансом.
   
Продемонстрируйте работу вашего приложения:
Программа должна обрабатывать все исключения с помощью конструкции try-catch, выводя соответствующие сообщения об ошибках.
